# Masinop — Privacy Policy site

This repository hosts the public **Privacy Policy** for the Masinop personal
finance and expense-tracking Android app, served via **GitHub Pages**.

- Live page: `https://<your-username>.github.io/masinop-privacy/`
- Source: [`index.html`](./index.html)

## How this is published

GitHub Pages is enabled on this repository (Settings → Pages → Deploy from a
branch → `main` / root). Any change pushed to `index.html` on `main` updates the
live page automatically after a short build.

## Updating the policy

Edit `index.html`, update the **Last updated** date near the top, commit, and
push. The change goes live within a minute or two.
